Tables

For the tables I made the SCSS handle all of the usage cases and will apropriately style the table based on the theme colors if used in conjunction with the themex-container class while also being responsive to the amount of size the parent div gives.

Head of Table
Name Features Price
Ipsit dolor, sit, amet, consectetur $98.07

Code Example:

                        
<table class='table'>
<caption>Head of Table</caption>
    <thead>
        <tr>
            <th scope='column'>Name</th>
            <th scope='column'>Features</th>
            <th scope='column'>Price</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th scope='row'>Ipsit</th>
            <td>dolor, sit, amet, consectetur</td>
            <td>$98.07</td>
        </tr>
    </tbody>
</table>
